# My Castle — 2D Portfolio Game

A 2D interactive portfolio built as a gothic castle game.
Explore the rooms, talk to the Guardian, find hidden bread
and discover the story of the developer behind it all.

Built with **Kaboom.js**, designed in **Tiled**, character created in **PixelLab**.

🎮 Live Demo: [YOUR_LINK_HERE]

## Features

- 2D top-down exploration with animated character
- NPC Guardian with quest system
- Collectible items (bread) that unlock map hints
- Interactive dialogues with portfolio info
- Gothic atmosphere: church music, fire ambience, particle effects
- Mobile friendly

## Stack

- Kaboom.js — game engine
- Tiled — map editor
- Vite — build tool
- Vanilla JavaScript

## How to run

`npm install` then `npm run dev`

## How to build

`npm run build`

## Character & Art

- Player character: created in PixelLab (chibi anime style)

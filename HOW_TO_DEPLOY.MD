# How to deploy (GitHub Pages)

1. In `vite.config.js` set base to your repo name:
   `base: "/2d-portfolio-kaboom"`

2. Run `npm run build`

3. Push to GitHub

4. Go to repository Settings → Pages → GitHub Actions

5. The game will be live at:
   `https://YOUR_USERNAME.github.io/2d-portfolio-kaboom/`
